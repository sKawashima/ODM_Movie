# ODM Movie

## Algorithm
1. js側で画像をロード
2. canvas#analyze(display:none)に画像を解像度をあわせて描写
3. canvas#analyzeの色情報を取得
4. いい感じに処理してcanvas#odmmに描写

## Top Technology
|||
|:-|:-|
|canvas|jsで制御できる画像処理空間|
|getImageData|canvasの描写内容をピクセルデータとして取得|
